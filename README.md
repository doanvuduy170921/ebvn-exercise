# Lesson 01 - Bookmark Service Health Check

This repository contains the solution for Assignment 1: Initializing an API server with a health-check endpoint using Clean Architecture principles.

## 🚀 Features
- **GET /health-check** endpoint.


## 🏃 How to Run and Test

This project uses a `Makefile` to automate tasks. Run the following commands in your terminal:

### Run the Application
```bash
make run
```
### Run Unit test
```bash
make test
```
### Run Integration test
```bash
make test-integration
